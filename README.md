# ParentLog — Your Parenting Companion

A repo-native AI vessel for parents. Track milestones, save memories, and get age-appropriate advice.

## Features
- 👶 Child profiles with developmental tracking
- 📅 Milestone logging (first smile, first steps, first words)
- 💾 Memory journal for precious moments
- 💡 AI-powered parenting advice
- 📆 Family schedule management

## The Cocapn Difference
ParentLog remembers everything. Every milestone. Every first. Every moment you want to hold onto. The cocapn lives in your vessel and grows with your family.

## Deploy
1. Fork this repo
2. Add your DeepSeek API key
3. Deploy to Cloudflare Workers
